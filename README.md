# Bot_helper

# 1 - Виртуальная клавиатура
Виртуальная клавиатура должна содержать выбор группы, к которой относиться студент. Потом же после выбора группы появятся кнопки с выбором дни недели, где уже будет находиться расписание.

# 2 - БД
Структура БД должна выглядеть таким образом:
id | id_user (telegram_user) | id_group | time | day_week

# 3 -  Функция бота
Бот в канале отправляет напоминания о занятии (за сутки, день недели и время). В личном сообщении пишет также, как и в канале.
